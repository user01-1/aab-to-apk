# .aab to .apk

## input
- .aab file
- .apks file path (browse_save_file)
- .apk file directory (browse_folder)
- keystore file
- keystore password
- key alias
- key password
<br>

## get keystore
- android studio install > build > genterate sign app bundle or APK > Androud APP bundle > create new
